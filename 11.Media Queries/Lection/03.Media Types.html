DESKTOP: @media screen and (min-width: 1024px) {...} TABLET: @media screen and (min-width: 768px) and (max-width: 1023px) {...}; SMARTPHONE: @media screen and (max-width: 767px) {...} Media Types describe the general category of a given device: - all (used for all media types devices); - print (used for printers); - screen (used for computer screens, tablets, smart-phones etc.); - speech (used for screen readers that "reads" the page out loud); The following media query will only set the body to 12px if the page is printed.It will not apply when the page is loaded in a browser: @media print {     body {     font-size: 12px;   } }
